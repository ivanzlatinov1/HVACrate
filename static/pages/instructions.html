Installing Instructions

Windows

Steps for installing HVACrate on Windows:

  • Install the app from the Download page
  • Extract the zippped folder
  • Run as administrator the executive (.exe)

    • * You may create a shortcut to the executive and place it on the Desktop

Linux

Steps for installing HVACrate on Linux:

  • Install the app from the Download page
  • Ensure the .AppImage file is executable on Linux: chmod +x ~/path-to-the-file/HVACrate-1.0.0-linux.AppImage
  • Double-click it or run ./HVACrate-1.0.0-linux.AppImage in bash
